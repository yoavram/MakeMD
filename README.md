MakeMD
===============================

version number: 0.0.1
author: Yoav Ram

Overview
--------

Assisting the production of PDF files from Markdown with Pandoc

Installation / Usage
--------------------

To install use pip:

    $ pip install makemd


Or clone the repo:

    $ git clone https://github.com/yoavram/makemd.git
    $ python setup.py install
